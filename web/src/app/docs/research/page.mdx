# Research and References

txray is built with transparent, inspectable methodology informed by public blockchain analysis research.

## Foundational Directions

- Address-linking heuristics and transaction graph analysis.
- Change output identification techniques.
- Wallet behavior pattern studies.
- Privacy limitations and adversarial counter-strategies.

## Documentation Principle

For every major heuristic, txray documents:

- what the heuristic assumes,
- when it works,
- when it fails,
- how strongly to trust the result.
