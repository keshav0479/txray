## What is txray?

txray provides three integrated tools for Bitcoin analysis:

- **Lens**: Decode transactions at the byte level.
- **Sherlock**: Cluster addresses and apply forensic heuristics.
- **Smith**: Build optimized PSBTs with better privacy and fee control.

## Quick Start

### Web Interface

1. Open the app.
2. Choose Lens, Sherlock, or Smith.
3. Paste transaction/block data.
4. Review the analysis.

### CLI

```bash
# Parse a transaction
cargo run --release -- parse tx <hex-or-file>

# Analyze a block
cargo run --release -- analyze <block-height>

# Build a PSBT
cargo run --release -- build <fixture.json>

# List famous blocks
cargo run --release -- famous
```

## Installation

```bash
git clone https://github.com/yourusername/txray.git
cd txray
cargo build --release
```
