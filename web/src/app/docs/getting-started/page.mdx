## What is txray?

txray provides three integrated tools for Bitcoin analysis:

- **Lens**: Decode transactions at the byte level.
- **Sherlock**: Cluster addresses and apply forensic heuristics.
- **Smith**: Build optimized PSBTs with better privacy and fee control.

## Quick Start

### Web Interface

1. Open the app.
2. Choose Lens, Sherlock, or Smith.
3. Paste transaction/block data.
4. Review the analysis.

### CLI

```bash
# Parse a transaction
txray parse tx fixtures/segwit_v0.json

# Analyze a block
txray analyze block.blk block.rev xor.dat

# Build a PSBT
txray build fixtures/simple_payment.json

# List famous blocks
txray famous
```

## Installation

```bash
git clone https://github.com/keshav0479/txray.git
cd txray
cargo build --release
```
