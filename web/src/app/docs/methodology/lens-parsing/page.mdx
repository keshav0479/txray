# Lens Methodology

Lens focuses on *transaction decoding and structure clarity*.

## What Lens Does

- Parses raw transaction data.
- Classifies script types.
- Extracts fee, weight, and format properties.
- Highlights potential anomalies.

## Philosophy

Lens is the "microscope" of txray: first understand bytes and structure, then move to inference.

## Output Style

Lens prioritizes readable, step-by-step breakdowns so users can verify each field.
